# Housing Affordability Analysis - DC Metro

Team: Join Genius  
Members: Leslie Rodriguez, Jennifer Reif, Juan Parada, Roger Rocha-Claros, Ryan Sollom

## Project Overview
This project analyzes housing affordability for young adults (ages 20–29) in the Washington, D.C. metropolitan area using IPUMS ACS data (2021-2023).

## Folder Structure
- `data/` : Raw and cleaned datasets
- `scripts/` : R scripts for data cleaning, analysis, and visualization
- `reports/` : Figures and white paper

## How to Run
1. Open R or RStudio
2. Run scripts in order: `01_data_cleaning.R` → `02_rent_burden.R` → `03_owner_burden.R` → `04_mapping_tool.R` → `05_visualizations.R`
